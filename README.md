## 🚀 Projetos em Destaque

### 🤖 [robo-dmae](https://github.com/RafaelLev/robo-dmae)
Automação completa para download e extração de contas de água do portal DMAE (Porto Alegre).
- Acessa o portal automaticamente via **Selenium**, baixa PDFs de múltiplas UCs
- Extrai dados (leitura, consumo, valor, vencimento) com **pdfplumber + regex**
- Preenche planilha Excel automaticamente com **openpyxl**
- Economiza 2 a 3 dias de trabalho manual por mês

`Python` `Selenium` `pdfplumber` `openpyxl` `Automação`

Plataforma de análise de investimentos para FIIs e ações brasileiras.
